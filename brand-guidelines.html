Components

Components should feel like a real mobile app: compact, touchable, and sectioned. Fluent-style icons carry the playful personality.

Header

Use the logo and app name without extra bevelled containers or decorative headers.

Game sections

Do / Don't

These rules keep Midday playful but production-ready.

Do

  • Use sections and hierarchy.
  • Keep primary actions easy to reach.
  • Use Fluent SVG/emoji assets for animals.
  • Let category colors communicate meaning.
  • Make parent screens calmer and more adult.

Don't

  • Fill the whole app with sunshine rays.
  • Use heavy bevels or clay-plastic shadows.
  • Show every game as an identical tile wall.
  • Create custom animal SVG drawings.
  • Overload children with competing animations.
